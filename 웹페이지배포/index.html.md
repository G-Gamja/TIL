- `<head>` : 문서의 메타데이터가 들어있는 태그로, 문서의 제목(title), 문자 인코딩 방식(charset), 뷰포트(viewport) 등의 정보가 정의됩니다.

- `<meta>` : 메타데이터를 정의하는 태그로, `<head>` 태그 안에 위치합니다. `charset`, `name`, `property`, `content` 등의 속성을 가집니다. 여기서는 문자 인코딩 방식(charset), 뷰포트(viewport), 웹 페이지와 관련된 많은 정보들이 설정되어 있습니다.

- `<title>` : 문서의 제목 태그로, 브라우저 탭에 표시되는 이름으로 사용됩니다.

- `<body>` : 실제 문서의 내용이 들어가는 태그로, 브라우저에 보여지는 콘텐츠가 모두 이 태그 안에 위치합니다. 이 HTML 코드에서는 JavaScript 로 만들어진 React App 이 실행될 DOM Element 가 정의되어 있습니다.

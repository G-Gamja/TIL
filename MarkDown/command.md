# 마크다운 문법 정리

텍스트 강조 및 체 변경(_이탤릭_,**강조**)
======
이텔릭체는 __*별표(asterisks)*__ 혹은 _언더바(underscore)_를 사용하세요.
두껍게는 **별표(asterisks)** 혹은 __언더바(underscore)__를 사용하세요.
**_이텔릭체_와 두껍게**를 같이 사용할 수 있습니다.
취소선은 ~~물결표시(tilde)~~를 사용하세요.
<u>밑줄</u>은 `<u></u>`를 사용하세요.
